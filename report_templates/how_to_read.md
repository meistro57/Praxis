# How to Read This Book

This report is divided into several parts to ensure full transparency and provenance:

1. **Part I — Program Summary**: Provides run provenance (models, prompt versions, configuration settings, collection snapshots), the program funnel showing how candidates progressed through safety gates, and distributions of practice modes and risk tiers.
2. **Part II — Approved Protocols**: Lists all generated experiment protocols. Each protocol includes the original Keystone statement, working hypothesis, steps, measurements, stop conditions, and interpretation limits.
3. **Part III — Observations and Reflections**: Contains structured logs and qualitative reflections for each observation completed by participants.
4. **Part IV — Candidates for Human Review**: Highlights adverse outcomes and compiled feedback recommendations.
5. **Appendices**:
   - *Appendix A — Non-Actionable Register*: Lists all Keystone candidates considered but refused, with detailed reasons.
   - *Appendix B — Safety Decision Log*: Lists deterministic safety validations.
   - *Appendix C — Failure Log*: Records errors or parsing failures.
   - *Appendix D — Configuration Snapshot*: Shows a redacted version of the run settings.

Ensure you review the **Epistemic Notice** before reading any protocols.
